# 🛢️ Crude Oil Price Analysis Project

## Project Overview
This project transforms raw monthly crude oil price data (Jan 1970 – Mar 2026) into a structured dataset for analytical insights. Key objectives include:

- Calculating Month-over-Month (MoM) changes and percentage returns  
- Classifying impact levels (HIGH / MID / LITTLE)  
- Identifying cycle phases (Recovery, Peak, Trough)  
- Preparing data for trend analysis, volatility detection, and strategy building  

## Dataset & Methodology
- **Source:** Kaggle (all ownership remains with the original creator)  
- **Transformations:**  
  - Expanded date into Month, Quarter, Year  
  - Calculated MoM and YoY changes  
  - Assigned impact categories and cycle phases  
- **Output:** 675 rows of cleaned and structured data ready for pivot tables, dashboards, and visualization
